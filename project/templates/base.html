Maximin Aware (MMA) - There is always somebody who gets no more than you

Tabs

Divide and Choose

Finds an item allocation for three people where one divides the items and gives the choice to his peers.

Finds an MMA1 allocation for 3 agents using approximate leximin n-partition.

MMA1 - Maximin aware by one item. If everyone gets more than you, at least it's by one item only.

A leximin n-partition maximizes the minimum value over all possible n partitions, and if there is a tie it selects the one maximizing the second minimum value, and so on.

We use approximate version of leximin n-partition, where it maximizes the minimum sum value of a bundle.

Alice, Bob, and Claire have some items they want to share. Each values the items differently. First, Claire splits the items to 3 bundles, using approximate leximin-n-partition, and lets Alice and Bob choose first. She keeps whatever is left after they decide. If there are any disputes among the two, Bob will split the bundles he prefers and allow Alice to select her bundle first.

Allocation by Matching

Finds an item allocation for any amount of people using maximum matching in graphs.

Finds an 1/2MMA allocation for any amount of agents using maximum weight matching in graphs and envy reduction procedure.

1/2MMA - Maximin aware by half. Your bundle will always be worth at least half as much as the remaining items.

A matching in a graph is a set of edges without common vertices. Maximum weight matching is a matching that maximises the weight of the edges.

The envy reduction procedure finds who is envious of whom. If there's a cycle of envy, it swaps the bundles so each envious person gets the bundle they want.

Alice, Bob, and Claire have some items they want to share. Each values the items differently. First, Claire splits the items to 3 bundles, using approximate leximin-n-partition, and lets Alice and Bob choose first. She keeps whatever is left after they decide. If there are any disputes among the two, Bob will split the bundles he prefers and allow Alice to select her bundle first.
